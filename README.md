# 🔄 Churn Prediction System

> A production-ready ML system to predict customer churn.

## Project Structure
```
churn-prediction/
├── src/          # All source code
├── data/         # Data (raw, processed)
├── models/       # Saved model artifacts
├── notebooks/    # EDA notebooks only
├── configs/      # Configuration files
└── tests/        # Unit and integration tests
```

```bash

cd churn-prediction
```

### 2. Set up environment
